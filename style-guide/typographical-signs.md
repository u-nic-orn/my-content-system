# Typographical symbols

Typographical symbols are often written incorrectly. A classic example are quotation marks, where double encoding characters `"` are frequently used. These can cause errors in web apps and must be explicitly excluded. This does not happen with correct typographical symbols. In addition, the correct symbols are more subtle and improve readability.

There are many different characters, which vary depending on the language. Below you'll find the symbols we use most frequently.

## Swiss quotation marks
Use the official Swiss quotation marks, the guillemets. These are the opening `«` and the closing `»`.

**Keyboard shortcuts (Swiss keyboard)**
- Mac: `⌥` + `,` and `⌥` + `⇧` + `,`
- Windows (numeric keypad): `Alt` + `174` and `Alt` + `175`

## English quotation marks
These are the opening `"` and the closing `"`.

**Keyboard shortcuts (Swiss keyboard)**
- Mac: `⌥` + `2` and `⌥` + `⇧` + `2`
- Windows: `Alt` + `0174` and `Alt` + `0148`

## Ellipsis
Use the dedicated ellipsis character `…` instead of three dots `...`.
When shortening sentences, add a space before the ellipsis.
If only a single word is shortened, you don't need a space.

Learn more in this [Duden entry](https://www.duden.de/sprachwissen/rechtschreibregeln/auslassungspunkte).

**Keyboard shortcuts (Swiss keyboard)**
- Mac: `⌥` + `.`
- Windows: `Alt` + `0133`

## Apostrophe
Use the correct apostrophe `'`. All others — such as `'` or `'` — are incorrect.

**Keyboard shortcuts (Swiss keyboard)**
- Mac: `⇧` + `⌥` + `!`
- Windows: `Alt` + `0146`

## En dash
When using a dash, use the en dash `–`. Do not use a hyphen.

**Keyboard shortcuts (Swiss keyboard)**
- Mac: `⌥` + `-`
- Windows: `Alt` + `0150`

## Backslash
This sneaky little character `\` can be surprisingly hard to find. Here is the correct keyboard shortcut.

**Keyboard shortcuts (Swiss keyboard)**
- Mac: `⌥` + `⇧` + `7`

## Non-breaking space
Insert a non-breaking space for `z. B.`. Written out, it is `zum Beispiel`, and the non-breaking space ensures that `z. B.` does not break across lines.

**Keyboard shortcuts (Swiss keyboard)**
- Mac: `⌥` + `Space`
