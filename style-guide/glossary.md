# Glossary

The master glossary lives in `master-glossary.csv` in this directory. It contains terminology in de-CH, fr-CH, it-CH, and en-GB, along with definitions, usage notes, and flags for translatability, case sensitivity, and department ownership.

## Using the glossary

- Always check the glossary before introducing new terminology in copy or UI
- The `don't` column lists terms to avoid
- The `comment` column provides usage context
- Cross-reference `related_terms` to stay consistent across the product

## Master glossary file

See [`master-glossary.csv`](./master-glossary.csv) for the full term list.
